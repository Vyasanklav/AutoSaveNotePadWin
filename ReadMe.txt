After unzipping the AutoSaver, you can run "AutoSaver For NotePad Win.bat" or its the shortcut-label "АвтоСейвер для Блокнота".
*For convenience, you can put the shortcut-label on the Desktop-table.

A folder for saving Notepad files will be created automatically in the same directory where located "AutoSaverForNotePadWin.bat."

nircmdc.exe - this is a special utility. She is needed for the AutoSaver's work. Don't tuch it, please.
