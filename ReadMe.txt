run "AutoSaver For NotePad Win.bat" or its
 the shortcut-label "АвтоСейвер для Блокнота".

*For convenience, you can put the shortcut-label on the Desktop
table.



A folder for saving Notepad files will be created 
automatically in the same directory where 
located "AutoSaverForNotePadWin.bat."
